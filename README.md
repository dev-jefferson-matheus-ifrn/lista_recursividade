# 📋 - Lista De Exercícios.

1. Implemente uma função recursiva que permita somar os elementos de um vetor de inteiros.
2. Implemente uma função recursiva contagemRegressiva(n) que imprime no console uma contagem de n até 0.
3. Crie uma função potencia(base, expoente) usando apenas multiplicações recursivas.
4. Implemente uma função recursiva que receba um número inteiro positivo n e calcule o somatório dos números de 1 a n.
5. A multiplicação de dois números inteiros pode ser feita através de somas sucessivas. Proponha um algoritmo recursivo multiplicação(n1, n2) que calcule a multiplicação de dois inteiros.
6. Implemente uma função recursiva para converter um número natural em base binária.
7. Implemente uma função recursiva para calcular o MMC (mínimo múltiplo comum) entre dois números.
8. Implemente uma função recursiva para dizer se uma palavra é palíndroma.
9. Pesquise como se pode desenvolver uma função de Fibonacci recursiva que seja eficiente e implemente essa solução.
10. Implemente uma função recursiva buscaBinaria(vetor, inicio, fim, alvo) que encontre o índice de um
elemento em um vetor já ordenado. Se o elemento não existir, a função deve retornar -1. A lógica
consiste em verificar o elemento do meio: (i) Se for o alvo, terminamos. (ii) Se o alvo for menor,
buscamos apenas na metade esquerda. (iii) Se o alvo for maior, buscamos apenas na metade direita.

# Passos Para Clonagem Do Repositorio E Consumo do Repositorio.

1. Clone O Repositorio Com o Seguinte Comomando
 ```Console

 ```
2. Mova para o Diretorio onde encontram-se o seu workspace(Eclipse, Netbeans, IntelliJ)
```Console
mv <repositorio> <workspace>
```
3. Abra com a sua IDE preferida
4. Abrir com Vscode
```Console
code <repositorio>
```
